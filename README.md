# Travel Agent App

Sistem manajemen layanan agen perjalanan yang komprehensif.

## Panduan Penggunaan untuk Admin

Berikut adalah langkah-langkah operasional untuk pengelola aplikasi:

1. **Login Admin**: Masuk ke aplikasi menggunakan akun yang memiliki kredensial role `admin`.
2. **Manajemen Paket Wisata**: 
   - Akses menu **"Manage Tour Packages"**.
   - Di sini Admin dapat membuat paket wisata baru, mengedit detail paket, atau menonaktifkan paket yang sudah tidak tersedia.
3. **Verifikasi Transaksi**:
   - Akses menu **"Manage Transactions"** untuk memantau pesanan masuk.
   - Periksa bukti pembayaran pelanggan dengan klik tombol **"View Proof"**.
   - Ubah status pesanan menjadi **Confirm (Paid)** jika pembayaran valid, atau **Cancel** jika tidak sesuai.
4. **Laporan Dashboard**:
   - Akses **"Admin Dashboard"** untuk melihat ringkasan performa bisnis, termasuk:
     - Total Revenue (Total Pendapatan)
     - Pesanan Sukses
     - Pesanan Pending
